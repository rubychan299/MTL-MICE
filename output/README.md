This directory contains the exported PDF figures included with the public
submission package.

## Contents

- Simulation figures generated from the scripts in `code/sim/summ/`.
- Real-data performance and transfer-network figures generated from
  `code/rda/plot_rda.R` and `code/rda/extract_id_rda.R`.

## Notes

- These files are the final figure artifacts used in the manuscript package.
- The large intermediate `.RData` objects produced by the original batch runs
  are not stored here.
